# 💳 Credit Card Fraud Detection

📌 Project Overview
This project addresses the critical challenge of identifying fraudulent financial transactions in a highly imbalanced environment. Using a dataset of 284,807 transactions with a fraud rate of only 0.17%, I developed a machine learning solution that prioritizes "Recall" to ensure 4 out of 5 fraudulent activities are captured, while providing a risk-based alert system for manual review teams.


## 🔍 Business Problem

Financial institutions face significant challenges in detecting fraudulent transactions due to highly imbalanced datasets and evolving fraud patterns. Traditional detection methods often fail to identify fraud accurately, leading to financial losses and increased operational costs.

---

## 🎯 Objective

To build a machine learning model that accurately detects fraudulent transactions and develop an interactive dashboard to monitor fraud patterns and support risk-based decision-making.

---

## 📊 Key Insights

* Dataset contained **284,807 transactions with extreme class imbalance (578:1)**
* Improved fraud detection recall from **61% to 81.6%** using SMOTE and model tuning
* Achieved **AUC score of 0.981** with Random Forest, outperforming baseline models
* Identified key transaction patterns contributing to fraudulent behavior

---

## 🚀 Business Impact

* Enables early detection of fraudulent transactions, reducing financial losses
* Improves risk assessment through a **4-tier fraud classification system (Low / Medium / High / Critical)**
* Reduces manual fraud review workload by approximately **60%**
* Supports data-driven fraud prevention strategies
